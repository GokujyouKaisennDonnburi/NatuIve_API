# 座標情報保護ルール

> 適用対象: `internal/**/*.go`（Go の実装コードを書くとき）。
> このファイルは CLAUDE.md から `@import` で常時ロードされる。

## 禁止パターン
- クライアントから受け取った緯度・経度をそのまま INSERT/UPDATE してはならない
- 座標を含む構造体を DB 層に直接渡す前に、geofuzzing 関数を経由させること
- レスポンスに raw 座標を含める場合、ユーザーの権限レベルに応じた公開範囲制御を必ず適用する

## チェックポイント
- `latitude`, `longitude`, `lat`, `lng`, `coord` を含むフィールドを DB に書き込むコードを見つけたら、geofuzzing 処理の有無を確認する
- 座標保護をバイパスする API エンドポイントが存在しないことを確認する
